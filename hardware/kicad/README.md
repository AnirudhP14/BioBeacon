# Kicad pcb files

This folder contains Ki-cad PCB files for
- BioBeacon ESP32 Wroom board
- BioBeacon sensor board - daughter board
- TB6612FNG motor board 

Additional ESP32 boards to be added to support additional ESP32 variants.
Separate TB6612FNG daughter board could provide more flexibility.
